# BLNT-CLI Project

This is a cross-platform AI-powered command-line interface built with Node.js.

## Features
- Local-first AI with Ollama
- API fallback to OpenAI
- Multiple interaction modes
- Beautiful terminal UI with neon theme

## Architecture
- Built with oclif framework
- Uses Ink for React-based terminal UI
- TypeScript for type safety
- Configurable via simple config commands
